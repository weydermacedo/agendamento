Endpoints API de Agendamentos

Lista todos os serviços. Opcionalmente, filtra por atendente.

Requisição:
GET {{api}}/api-agendamento.php?servicos&atendente={{atendente}}
Resposta:
[
    { "id": 1, "nome": "Serviço 1" },
    { "id": 2, "nome": "Serviço 2" }
]

Lista todos os horários disponíveis.

Requisição:
GET {{api}}/api-agendamento.php?horarios?atendente={{atendente}}
Resposta:
[
    { "id": 1, "horario": "09:00" },
    { "id": 2, "horario": "10:00" }
]

Lista todos os atendentes. Opcionalmente, filtra por disponibilidade.

Requisição:
GET {{api}}/api-agendamento.php?atendentes?somente_disponiveis=1
Resposta:
[
    { "id": 1, "nome": "Atendente 1", "cargo": "Cargo 1", "disponivel": 1 },
    { "id": 2, "nome": "Atendente 2", "cargo": "Cargo 2", "disponivel": 0 }
]

Agenda um horário.

Requisição:
GET {{api}}/api-agendamento.php?agendar&nome={{nome}}&telefone={{telefone}}&servico={{servico}}&data={{data}}&horario={{horario}}&atendente={{atendente}}
Resposta:
{ "status": "success", "message": "Agendamento realizado com sucesso!" }

Consulta agendamentos pelo telefone.

Requisição:
GET {{api}}/api-agendamento.php?consultar?telefone={{telefone}}
Resposta:
[
    { "id": 1, "nome": "Nome", "telefone": "999999999", "data_hora": "2023-10-12 09:00", "servico": "Serviço", "atendente": "Atendente" }
]

Cancela um agendamento.

Requisição:
GET {{api}}/api-agendamento.php?cancelar?telefone={{telefone}}&data={{data}}
Resposta:
{ "status": "success", "message": "Agendamento cancelado com sucesso!" }
